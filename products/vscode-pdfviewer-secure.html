VSCode PDF Viewer Secure

Security-first, offline-first PDF viewing for VS Code

A hardened PDF viewer extension for Visual Studio Code, designed for business and enterprise environments. VSCode PDF Viewer Secure prioritizes security and privacy, offering robust protection for sensitive documents and workflows.

Key Features

  • Offline-first PDF viewing
  • Read-only by default
  • Restrictive webview Content Security Policy (CSP)
  • PDF scripting disabled
  • External links disabled by default
  • No shell or child-process execution
  • Feature flags for controlled enablement

Security Posture

VSCode PDF Viewer Secure is engineered to minimize attack surface and support controlled business environments. Features such as printing, downloading, opening files, forms, and annotation editing are disabled by default. The extension uses a bundled viewer and local assets, ensuring document handling remains secure and isolated.
